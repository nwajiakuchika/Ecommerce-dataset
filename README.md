**E_COMMERCE SALES ANALYSIS**

*Project Overview*

This analyzed an Ecommerce dataset to uncover business insights,sales trends and profits performance.
This analysis include data cleaning,Exploratory Data Analysis (EDA,visualization and machine learning
for profit.
**DATA FEATURES**

- Order Date,- Category,- Region,- Quantity,- Product Name,- Sales and Profit

- 
**TOOLS USED**
- Python,- Matplotlib,- Seaborn,- Numpy,- Pandas,- Scikit-Learn
**MACHINE LEARNING**

  
A RandomForestRegressor was used to predict profit based on business related
features
**KEY INSIGHTS**

Some categories generated high profit than others.
Regional performance varied across sales location.
QUantity purchase influence profit and revenue.
Sales trends help identify profitable business opportunity.
**CONCLUSION**
This project demostrate how data analysis and machine learning 
can help businesses understand customer purchasing behavior,and improve
decision making.
